AutoShare Privacy Policy

Effective date: May 20, 2026

AutoShare is a local network file sharing app. We built this app to let users transfer files directly between nearby devices over local Wi-Fi.

1. Data We Collect

AutoShare does not collect personal data to remote servers.

The app can store the following data locally on your device:

  • Device name and randomly generated device ID
  • Paired device list and pairing tokens
  • Files you receive and manage through the app
  • App preferences (for example port and settings)

2. Permissions We Use

  • Network / Wi-Fi permissions: used to discover nearby devices and transfer files over local network.
  • Storage permissions (including all files access when granted): used for browsing, moving, opening, and receiving files.
  • Notification permission: used to notify file transfer progress and completion.
  • Install packages permission: used only when you tap an APK file and choose to open Android native installer.

3. How We Use Data

Local data is used only to provide core app functionality:

  • Device discovery and pairing
  • Secure local file transfer
  • File management operations
  • Transfer notifications

4. Data Sharing

We do not sell data and we do not share personal data with third-party advertisers. Files are transferred only between devices you connect through your local network.

5. Data Security

AutoShare is designed for local transfers. You are responsible for the security of your network and for pairing only trusted devices.

6. Children’s Privacy

AutoShare is not directed to children under 13. We do not knowingly collect personal information from children.

7. Changes to This Policy

We may update this policy from time to time. Any updates will be posted on this page with a revised effective date.
